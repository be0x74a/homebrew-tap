# be0x74a/homebrew-tap

Homebrew formulae for kubectl plugins by be0x74a.

## Setup

```bash
brew tap be0x74a/tap
```

## Available formulae

| Formula | Description |
|---------|-------------|
| `kubectl-xctx` | Execute kubectl commands across multiple contexts matching a regex |

## Install

```bash
brew install --cask be0x74a/tap/kubectl-xctx
```

## Update

```bash
brew update
brew upgrade --cask kubectl-xctx
```
